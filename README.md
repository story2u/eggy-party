# 🥚 蛋仔派对 Demo

> 用 Python + Pygame 开发的 2D 派对闯关小游戏  
> 适合初学者学习游戏开发

---

## 🎮 游戏介绍

这是一款模仿《蛋仔派对》的 2D 平台跳跃游戏 demo：

- 🏃 控制可爱的蛋仔角色
- 🏆 与 AI 对手竞速闯关
- ⚡ 躲避旋转锤子、尖刺等障碍
- 🎯 到达终点，争夺第一

### 操作方式

| 按键 | 动作 |
|------|------|
| ← → 或 A D | 左右移动 |
| ↑ 或 W 或 空格 | 跳跃（支持二段跳） |
| R | 重新开始 |
| ESC | 退出游戏 |

---

## 🔧 技术栈

| 技术 | 用途 |
|------|------|
| Python 3.12 | 编程语言 |
| Pygame 2.6 | 游戏引擎（渲染、输入、音效） |

### 为什么选这些？

- **Python**：语法简单，适合初学者
- **Pygame**：2D 游戏最友好的入门库，不需要学复杂引擎
- **没有 AI API**：这个 demo 是纯本地游戏，让你先理解游戏是怎么跑起来的

## 🚀 如何运行

```bash
# 1. 进入项目目录
cd eggy-party

# 2. 安装依赖
pip3 install pygame --break-system-packages

# 3. 运行游戏
python3 main.py
```

## ✅ 如何测试

项目使用 Python 标准库 `unittest` 做 TDD 回归测试，不需要额外安装测试框架。

```bash
python3 -m unittest discover -s tests -v
```

每个测试方法的 docstring 都记录了对应功能的设计意图，方便回归失败时理解当初为什么要这样设计。

## 🎯 关卡设计

### 关卡 1：新手草原 🌿
- 简单的平台跳跃
- 少量旋转锤
- 让玩家熟悉操作

### 关卡 2：齿轮工厂 ⚙️
- 更多的移动障碍
- 移动平台
- 需要更精准的操作

### 关卡 3：最终乱斗 🔥
- 最难的障碍组合
- 更长的赛道
- 真正的挑战

### 关卡 4：一二三木头人
- 红灯停、绿灯跑
- 红灯时移动会被淘汰

## 🧠 代码设计思路（写给想学习的你）

### 游戏循环（Game Loop）
```
每秒运行 60 次：
  1. 处理输入（键盘按键）
  2. 更新位置（物理计算、碰撞检测）
  3. 渲染画面（画角色、平台、障碍物）
```

### 物理系统（简化版）
```
每个角色每帧：
  - 重力让速度向下增加
  - 碰到平台就站在上面
  - 碰到障碍物就扣血
```

### AI 对手逻辑
```
每帧：
  - 如果前方有坑，就跳跃
  - 如果前方有障碍物，找时机躲避
  - 朝终点方向移动
```

## 🛣️ 下一步你可以学什么？

学完这个 demo 后，你可以继续：

1. **加图片** → 把圆形换成真正的蛋仔图片
2. **加音效** → 跳跃声、碰撞声、胜利音乐
3. **联机对战** → 学习网络编程，朋友联机一起玩
4. **地图编辑器** → 让玩家自己设计关卡
5. **AI 生成关卡** → 调用大模型自动生成关卡
