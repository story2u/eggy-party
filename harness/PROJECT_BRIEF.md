# Project Brief

## 项目定位

`eggy-party` 是一个 Python + Pygame 实现的 2D 派对闯关小游戏 demo，灵感来自《蛋仔派对》。它的核心价值是教学和原型验证，不是大型游戏工程。

## 目标体验

- 玩家控制一个蛋仔角色，在横向平台关卡中移动、跳跃、到达终点。
- AI 对手自动向终点移动，并尝试跳过坑和障碍。
- 关卡逐步增加难度，包括旋转锤、尖刺、移动平台、一二三木头人。
- 游戏应该能本地直接启动，不依赖网络服务。

## 技术边界

- 语言：Python 3。
- 渲染和输入：Pygame。
- 当前没有资源管线，主要用 Pygame 图形 primitives 绘制。
- 当前没有自动化测试框架，验证以编译检查和 Pygame smoke test 为主。
- 当前没有打包流程。

## 非目标

- 不引入大型 ECS、物理引擎或复杂状态管理，除非明确需要。
- 不把 demo 改成联网游戏，除非任务明确要求。
- 不用图片资源替换全部绘制逻辑，除非任务明确要求。
- 不为了单个小功能做大规模目录重构。

## 用户和维护者

- 主要用户：学习 Python/Pygame 的初学者。
- 主要维护者：AI coding agent + 人类审阅。
- 文档语言以中文为主，代码保持清晰直观。
